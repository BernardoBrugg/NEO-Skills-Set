# Claude Code - AI_RAG

Leia `AGENTS.md` primeiro. Para tarefas complexas, use `/confidence-check` primeiro e depois escolha a skill adequada em `.agents/skillset.md`.

## Uso recomendado

- Use subagents para revisao, pesquisa, avaliacao e auditoria de safety quando a tarefa for grande.
- Leia prompts, configs de modelo, retrieval, ingestao e testes relevantes antes de editar.
- Nao mude modelo, chunking, embedding ou prompt de producao sem registrar impacto esperado.
- Antes de terminar, rode testes e avaliacoes disponiveis.

## Prompt inicial sugerido

`Leia AGENTS.md e .agents/skillset.md. Depois implemente a tarefa seguindo as regras do workspace.`
