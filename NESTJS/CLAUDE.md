# Claude Code - NESTJS

Leia `AGENTS.md` primeiro. Para tarefas complexas, use `/confidence-check` primeiro e depois escolha a skill adequada em `.agents/skillset.md`.

## Uso recomendado

- Use subagents para revisao, pesquisa, testes ou auditoria quando a tarefa for grande.
- Leia module, controller, service, DTOs e providers relevantes antes de editar.
- Preserve dependency injection e escopos de providers.
- Antes de terminar, rode testes unitarios, e2e ou build quando existirem.
