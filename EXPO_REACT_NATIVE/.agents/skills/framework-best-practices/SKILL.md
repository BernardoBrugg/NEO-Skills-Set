# Expo + React Native Best Practices

- Read navigation setup, screen structure and Expo config before editing.
- Consider iOS, Android, safe area, keyboard, small screens and accessibility.
- Preserve loading, error, empty and success states.
- Avoid heavy unvirtualized lists.
- Check Expo SDK compatibility before adding dependencies.
- Treat native permissions and prebuild changes as sensitive.
